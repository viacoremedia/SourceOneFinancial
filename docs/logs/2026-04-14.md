## Discoveries

- **Missing `--bg-card` token was root cause of DigestPanel looking stuck in dark mode.** Multiple component CSS files referenced `var(--bg-card, #162031)` but the variable was never defined in `tokens.css`. This meant the hardcoded navy fallback was always used, completely bypassing the light theme override system. Adding it to both `:root` and `[data-theme="light"]` fixed the entire Daily Digest panel in one shot.
- **Accent color choice cascading failures.** When `--accent-primary` was changed to dark slate (`#0F172A`) for a "modern monochrome" look, it caused the sorted column header indicators (`.thSorted`) to become invisible against the dark text — effectively making the entire sort system vanish. Had to revert to a distinct Royal Blue (`#2563EB`).
- **Theme-aware heatmap required an architectural decision.** The original HSLA interpolation-based heatmap looked good on dark backgrounds but produced muddy, low-contrast pastels on white. Rather than trying to tune the alpha values, the clean solution was early-returning solid hex text colors for light mode — no backgrounds at all.
